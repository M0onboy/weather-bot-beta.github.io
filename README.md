# Telegram Weather Bot + Mini App

Готовый Python-проект для Telegram-бота с прогнозом погоды, определением местоположения, подробной статистикой и статичным Telegram Mini App в стиле Apple Weather.

## Что умеет

- Принимает геолокацию из Telegram.
- Ищет погоду по названию города.
- Показывает текущую погоду, прогноз на 7 дней и подробную статистику.
- Открывает Telegram Mini App через кнопку.
- Mini App можно разместить на GitHub Pages.
- Погодные данные берутся из Open-Meteo, API-ключ для погоды не нужен.

## Быстрый запуск бота

1. Создайте бота через BotFather и получите `BOT_TOKEN`.
2. Скопируйте `.env.example` в `.env`.
3. Заполните:

```env
BOT_TOKEN=ваш_токен_бота
MINI_APP_URL=https://your-username.github.io/your-repo/
DEFAULT_CITY=Almaty
```

4. Установите зависимости:

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

5. Запустите:

```bash
python -m weather_bot
```

Если зависимости установлены в локальное окружение проекта, можно запустить так:

```bash
.venv\Scripts\python.exe -m weather_bot
```

Или откройте файл `start_bot.bat` двойным кликом. Окно нужно оставить открытым, пока бот должен отвечать в Telegram.

## Публикация Mini App на GitHub Pages

1. Создайте GitHub-репозиторий.
2. Загрузите папку `miniapp` в репозиторий.
3. В настройках репозитория откройте `Pages`.
4. Выберите публикацию из ветки `main`, папка `/miniapp`.
5. Скопируйте URL GitHub Pages в `MINI_APP_URL`.

После этого команда `/app` в боте будет открывать Mini App.

В BotFather можно дополнительно настроить кнопку меню:

```text
/setmenubutton
```

Затем выберите бота, укажите название кнопки и тот же HTTPS URL GitHub Pages.

Для Telegram Mini App обязательно используйте HTTPS. GitHub Pages уже выдает HTTPS-ссылку.

## Команды

- `/start` - приветствие и меню.
- `/weather` - запросить геолокацию или город.
- `/forecast` - прогноз по последней выбранной локации.
- `/stats` - подробная статистика по последней выбранной локации.
- `/hourly` - почасовой прогноз на ближайшие часы.
- `/advice` - краткий совет на день.
- `/alerts` - погодные предупреждения.
- `/favorites` - избранные города.
- `/app` - открыть Telegram Mini App.
- `/lang` - выбрать язык интерфейса: русский, казахский или английский.
- `/notify` - включить, выключить или проверить ежедневные уведомления о погоде.

Можно также просто отправить название города, например `Astana`.

Для уведомлений сначала отправьте город или геолокацию, затем используйте `/notify` и нажмите «Включить». Бот будет отправлять ежедневный прогноз примерно в 08:00 по часовому поясу выбранной локации, если он известен.

## SQLite простыми словами

SQLite - это локальная база данных в одном файле. Для неё не нужен отдельный сервер: Python сам создаёт файл `data/weather_bot.sqlite3` и хранит там пользователей, язык, последнюю локацию, избранные города и уведомления.

Это удобнее JSON, потому что данные разделены по таблицам и их проще расширять. Например, избранные города лежат отдельно от уведомлений, а настройки пользователя отдельно от прогноза.

Файл базы добавлен в `.gitignore`, поэтому пользовательские данные не попадут на GitHub.

## Где менять внешний вид Mini App

- `miniapp/index.html` - структура экрана.
- `miniapp/styles.css` - визуальный стиль.
- `miniapp/app.js` - загрузка погоды, геолокация, статистика.

Mini App полностью статичный: отдельный сервер для него не нужен.
